FR-111  The editor shall allow users to edit the generated Transon JSON directly and sync accepted edits back to the workspace.

FR-112  Direct JSON edits shall be applied to the workspace only when the edited text is valid JSON and within the supported surface.

FR-113  While a direct JSON edit is unparsed or rejected, the editor shall not silently modify the workspace.

FR-055  The editor shall report ambiguous, zero, or partial variant matches rather than silently rewriting the template.

FR-131  An accepted edit syncs to the workspace without rewriting the panel text while the editor retains focus.
