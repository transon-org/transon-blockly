The problem

Every system speaks JSON — never the same JSON.

the API you call
{ "countryCode": "FR",
  "countryName": "France" }
≠
the config you load
{ "countries": { "FR": "France" },
  "locale": "fr-FR" }
≠
the service you feed
[{ "name": "France",
   "code": "FR" }]
so we reshape it, all day — with throwaway code, or a one-liner
